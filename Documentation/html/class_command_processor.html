void CommandProcessor::process_command ( std::string & input)

Parses and processes an input command string.

Tokenizes the input, finds the matching registered command by name and argument count, and executes it. Throws an exception if the command is unknown or argument count mismatches.

Parameters
inputThe raw input string containing command and arguments.
Exceptions
std::runtime_errorif the command name is not found or argument count is incorrect.
