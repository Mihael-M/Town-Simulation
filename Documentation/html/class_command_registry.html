void CommandRegistry::register_all_commands ( CommandProcessor & processor)

Registers all commands into the given CommandProcessor.

This method sets up all available commands by associating command names and expected argument counts with their corresponding command objects, which handle the execution logic.

Parameters
processorThe CommandProcessor instance to register commands with.
