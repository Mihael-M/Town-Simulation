virtual void Command::execute ( const std::vector< std::string > & args)
pure virtual

Executes the command with the provided arguments.

This pure virtual method must be implemented by all concrete command classes. It defines the behavior that occurs when the command is invoked with arguments.

Parameters
argsA vector of strings representing the arguments passed to the command.
